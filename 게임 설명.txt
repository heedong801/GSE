==================
스테이지 ( 4개의 웨이브로 구성 )

1스테이지 : 악마파리, 성큰, 거미
2스테이지 : 성큰, 미라파리
3스테이지 : 좀비성큰, 거미
4스테이지 : 보스
==================

거미 : 무작정 플레이어를 쫒아온다
악마파리, 미라파리 : 생성된 지점에서 원 운동을 하다 피격시 플레이어 추적
성큰 : 2가지의 공격 패턴 ( 십자로 총쏘기, 엑스자로 총쏘기 ) 번갈아 사용
좀비성큰 : 한번에 십자 + 엑스자로 총쏘기

보스 : 플레이어 겨냥 총쏘기, 몬스터 소환
==================
GameEnd 조건 : 플레이어의 사망, 보스의 사망
